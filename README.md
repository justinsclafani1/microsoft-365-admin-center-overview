<h1>Microsoft 365 Admin Center Overview</h1>

<h2>Roles, Permissions, and Password Reset</h2>

This project demonstrates common Microsoft 365 administrative tasks using the Microsoft 365 Admin Center. The project covers assigning roles and permissions, resetting user passwords, and verifying successful user access.

<h2>Prerequisites</h2>

- Microsoft 365 account with administrator access to the Microsoft 365 Admin Center

<h2>Task 1 – Sign in to Microsoft 365 and Assign Roles</h2>

<h3>1. Access Microsoft 365 Admin Center</h3>

<p>
Access Microsoft 365 and navigate to the Admin Center.
</p>

<b>Instructions</b>

<p>
1. Sign in to Microsoft 365 using an administrator account.<br />
2. Open the Microsoft 365 Admin Center.<br />
3. Review the administrative dashboard.<br />
4. Locate common administrative sections, including:<br />
&nbsp;&nbsp;&nbsp;&nbsp;• Users<br />
&nbsp;&nbsp;&nbsp;&nbsp;• Active Users<br />
&nbsp;&nbsp;&nbsp;&nbsp;• Roles<br />
&nbsp;&nbsp;&nbsp;&nbsp;• Settings<br />
</p>

<h3>2. Assign Administrative Roles</h3>

<p>
Assign administrative permissions to a Microsoft 365 user account.
</p>

<b>Instructions</b>

<p>
1. Navigate to <b>Users</b> → <b>Active Users</b>.<br />
2. Select the desired user account.<br />
3. Open <b>Manage Roles</b>.<br />
4. Choose the appropriate administrative role.<br />
5. Review the assigned permissions.<br />
6. Save the changes.<br />
</p>

<h2>Task 2 – Reset Password and Verify User Sign-In</h2>

<h3>1. Reset User Password</h3>

<p>
Reset the password for an existing Microsoft 365 user account.
</p>

<b>Instructions</b>

<p>
1. Navigate to <b>Users</b> → <b>Active Users</b>.<br />
2. Select the user account requiring a password reset.<br />
3. Select <b>Reset Password</b>.<br />
4. Choose a password option:<br />
&nbsp;&nbsp;&nbsp;&nbsp;• Automatically generate a password<br />
&nbsp;&nbsp;&nbsp;&nbsp;• Manually specify a password<br />
5. Enable <b>Require password change at next sign-in</b> if needed.<br />
6. Save the password changes.<br />
</p>

<h3>2. Verify User Sign-In</h3>

<p>
Confirm that the password reset was successful by signing in with the updated credentials.
</p>

<b>Instructions</b>

<p>
1. Sign in using the test user account.<br />
2. Enter the newly assigned password.<br />
3. Complete the password change if prompted.<br />
4. Confirm successful access to Microsoft 365.<br />
</p>
